# DAM Properties

Luxury real estate platform for **Obour City, Egypt** — built with Next.js 16, React 19, TypeScript, Tailwind CSS 4, and Framer Motion.

## Run

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Pages

| Route | Description |
|-------|-------------|
| `/` | Cinematic homepage with all sections |
| `/properties` | Property listings + filters |
| `/properties/[slug]` | Property detail page |
| `/compare` | Side-by-side comparison (up to 4) |
| `/blog` | Insights & articles |
| `/contact` | Contact form |
| `/admin` | Premium admin dashboard shell |

## Features

- Cinematic hero with video background & glass search
- Natural-language live property search
- Interactive Obour City district map
- AI property matcher chat (demo)
- AI valuation & mortgage calculator
- Favorites & compare (localStorage)
- Arabic / English toggle + RTL
- Custom cursor, loading screen, scroll animations
- Investment dashboard with charts (Recharts)

## Stack

Next.js 16 · React 19 · TypeScript · Tailwind 4 · Framer Motion · Recharts · Lucide
